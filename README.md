## 🖥️ Run Locally

### Option 1 — Direct Open (Simplest)
Just double-click `index.html` — opens in your browser instantly. No install needed.

### Option 2 — Python Server (Recommended)
```bash
# Make sure Python is installed, then run:
python server.py

# OR use Python's built-in server:
python -m http.server 3000

# Open browser at:
http://localhost:3000
```

### Option 3 — VS Code Live Server
1. Install **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. Auto-opens at `http://127.0.0.1:5500`

## 🚀 Deploy to GitHub Pages (Free Hosting)

```bash
# 1. Clone or init repo
git init
git add .
git commit -m "🚀 Initial commit"

# 2. Push to GitHub
git remote add origin https://github.com/sachinsolanki18/sachin-portfolio.git
git branch -M main
git push -u origin main
```

Then go to:
**GitHub Repo → Settings → Pages → Source: main / root → Save**

Your live URL:
```
https://sachinsolanki18.github.io/sachin-portfolio
```
